# Unity 3D Endless Runner

## Project Description
This is a team-based endless runner game created for the NTU module **IM2073**. The game is built in a 3D environment using **Unity** and **C#**, where the player character continuously runs forward while avoiding obstacles and collecting items. The environment and challenges are generated dynamically to create an engaging infinite running experience.

## Unity Version
This project was developed using **Unity version 2022.3.17f1**.

## Tutorial Reference
This game was developed with guidance from a Unity endless runner tutorial playlist on YouTube:

https://youtube.com/playlist?list=PLZ1b66Z1KFKiR-IVN688Jo0BbOh4aV6Dt

This playlist walks through building movement, spawning systems, obstacle logic, and game mechanics for a 3D endless runner in Unity.

## How to Run
1. Open this project in **Unity Hub**.
2. Open the main scene from the `Assets` folder.
3. Press **Play** to run the game in the Unity Editor.
